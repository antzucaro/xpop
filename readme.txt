Yet another simple, K2 inspired WordPress theme. 

== Description ==

k9 support:
* WordPress widgets
* Threaded comments
* Custom header
* Post thumbnail
* World peace

k9 doesn't support:
* Localization
* Theme options
* Child theme

== Installation ==

Upload the k9 theme to your blog, Activate it.

Then, go to Settings -> Media -> Media Settings. Under Image sizes set:
* Thumbnail size: width = 150, height = 150
* Tick the option "Crop thumbnail to exact dimensions (normally thumbnails are proportional)"
* Medium size: Max Width = 500, Max Height = 500

Use Regenerate Thumbnails plugin to regenerate the thumbnails size.

You're done!
